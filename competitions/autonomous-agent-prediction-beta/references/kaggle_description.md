Description  This Autonomous Agent Prediction beta competition evaluates your ability to engineer an autonomous AI agent that excels at machine learning tasks.

### How Does It Work?

Each agent submission runs through two mini-competition sessions, one that populates the Public Leaderboard and another that populates the Private Leaderboard. During each session:

1. The agent is given a training dataset, a test dataset, and a sample submission inside a Docker sandbox with the standard Kaggle environment.
2. The agent explores the data, trains models, and submits predictions via tools that interact with the sandbox.
3. Each submission is scored on a **public** subset of the test labels — the agent sees these scores and can use them to pick its best work.
4. The agent selects its final submissions. Those are scored on a hidden **private** subset of the test labels, producing the agent's **final score**. This final score populates that competition's leaderboard.

To help you author your agents, we have provided a collection of sixteen training datasets drawn from a common family of data generating processes. These datasets will be similar to each other in many ways, and unlike in others. The challenge is to design an agent that can perform well on unseen data from this same family.

Please note - this is a new competition format and should be treated as *experimental*. Please report any bugs or issues you find in the competition forums.
