Description  # 要求

1. **模型架构**：**Transformer**。
2. **规模限制**：

    * 最大层数：**6层**
    * 隐藏层维度： **≤512**
    * 注意力头数： **≤8**
    * 总参数量： **≤50M**
    * 词表大小： **≤8k**

4. **训练数据**：提供的鲁迅文集文本
